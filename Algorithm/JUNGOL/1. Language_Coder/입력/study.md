### 513:입력 - 자가진단5

```python
yd_result = round(2.1 * 91.44, 1)
print("%4syd = %5scm" % ('2.1', str(yd_result)))
in_result = round(10.5 * 2.54, 1)
print("%4sin = %5scm" % ('10.5', str(in_result)))

# round(실수,n) -> 소수점을 n번째 까지만 표현하고 반올림
```
